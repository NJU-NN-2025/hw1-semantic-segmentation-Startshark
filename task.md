# 作业内容

语义分割（semantic segmentation）是一项计算机视觉任务，用于将图像划分成多个区域，每个像素都被标注为特定的类别（如道路、树木、车辆等）。在目标检测问题中，我们一直使用方形边界框来标注和预测图像中的目标。语义分割问题重点关注于如何将图像分割成属于不同语义类别的区域。

语义分割可以识别并理解图像中每一个像素的内容：其语义区域的标注和预测是像素级的。其应用领域非常广泛，包括自动驾驶（识别道路、行人等）、医学影像（分割器官或病变区域）、以及遥感图像分析（区分地形、植被等），为许多领域提供了更精确的视觉感知能力。
